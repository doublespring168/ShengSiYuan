Android AIDL

---------

RPC: Remote Procedure Call, 远程过程调用, RPC框架很多都是跨语言的

1. 定义一个接口说明文件(IDL), 描述了对象(结构体)、对象成员、接口方法等一系列信息

2. 通过RPC框架所提供的编译器将接口说明文件编译成具体语言文件

3. 在客户端与服务器端分别引入RPC编译器所生成的文件即可像调用本地方法一样调用远程方法
